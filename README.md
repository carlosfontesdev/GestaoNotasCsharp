# Gestão de Notas Escolares – C# Windows Forms

Sistema avançado de registo e análise de notas com interface gráfica completa.

Desenvolvido durante a formação de **Programação de Informática** – IEFP Braga (2025–2026)

### Funcionalidades
- Registo de alunos (nome, freguesia, ano, sexo)
- 12 disciplinas com notas 0–20
- Cálculo automático da média final
- Contagem de negativas
- Colorização condicional das células (verde/amarelo/vermelho)
- Estado final automático: Aprovado / Reprovado
- Código estruturado com métodos separados

### Tecnologias
- C# .NET
- Windows Forms + DataGridView
- Visual Studio 2022/2023
